# 🎬 ScreenMatch POO

Projeto desenvolvido durante meus estudos de Java com foco em Programação Orientada a Objetos (POO).

Este repositório faz parte da minha jornada de aprendizado em Java e tem como objetivo aplicar conceitos de orientação a objetos através da evolução do projeto ScreenMatch.

## 📚 Conceitos que serão praticados

* Classes e Objetos
* Atributos e Métodos
* Encapsulamento
* Herança
* Polimorfismo
* Coleções
* Boas práticas de programação

## 🛠️ Tecnologias utilizadas

* Java
* IntelliJ IDEA
* Git
* GitHub
* Linux Ubuntu

## 🚀 Objetivo

Desenvolver uma aplicação inspirada em plataformas de filmes e séries, aplicando gradualmente os conceitos de Programação Orientada a Objetos.

## ▶️ Como executar

1. Clone o repositório.
2. Abra o projeto no IntelliJ IDEA.
3. Execute a classe principal da aplicação.
